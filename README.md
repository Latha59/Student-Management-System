# Student Management Systed

## Overview

Sp_Users is a Java Spring Boot application developed to manage user details efficiently. This project demonstrates the use of Spring Boot, Maven, REST APIs, and database integration for performing user management operations.

## Features

* Add new users
* View all users
* Search user by ID
* Update user details
* Delete users
* RESTful API implementation
* Database connectivity using Spring Data JPA

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Maven
* MySQL / H2 Database
* REST API

## Project Structure

* `src/main/java` → Java source files
* `src/main/resources` → Application properties and static resources
* `pom.xml` → Maven dependencies and build configuration

## How to Run

1. Clone the repository
2. Open the project in STS / Eclipse / IntelliJ IDEA
3. Configure database in `application.properties`
4. Run the main Spring Boot application
5. Access APIs using Postman or browser

## Sample API Endpoints

* `POST /users` → Add user
* `GET /users` → Get all users
* `GET /users/{id}` → Get user by ID
* `PUT /users/{id}` → Update user
* `DELETE /users/{id}` → Delete user

This project was created for learning and practice in Java Full Stack Development and Spring Boot backend development.
